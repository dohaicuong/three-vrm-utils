---
title: Getting VRM Animations
description: Where to find or create VRMA animation files for your project.
---

VRMA is the animation format for VRM models. You can find and create VRMA files from the following platforms:

- <a href="https://hub.vroid.com/en" target="_blank" rel="noopener noreferrer">VRoid Hub</a> — Browse and download VRMA animations shared by the community.
- <a href="https://editor.librn.com/" target="_blank" rel="noopener noreferrer">Librn Editor</a> — Create and edit VRMA animations in the browser.

For more platforms, see the <a href="https://vrm.dev/en/showcase/?flags=98" target="_blank" rel="noopener noreferrer">VRM showcase list</a>.

## FBX Animations

- <a href="https://www.mixamo.com/" target="_blank" rel="noopener noreferrer">Mixamo</a> — Browse and download FBX animations from Adobe's library.
- <a href="https://github.com/tk256ailab/fbx2vrma-converter" target="_blank" rel="noopener noreferrer">fbx2vrma-converter</a> — Convert FBX animations to VRMA format.
