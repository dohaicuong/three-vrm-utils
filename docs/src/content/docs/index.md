React hooks and utilities for working with <a href="https://vrm.dev/en/" target="_blank" rel="noopener noreferrer">VRM</a> avatars in [React Three Fiber](https://github.com/pmndrs/react-three-fiber). Extension support for [@pixiv/three-vrm](https://github.com/pixiv/three-vrm) and [@pixiv/three-vrm-animation](https://github.com/pixiv/three-vrm/tree/dev/packages/three-vrm-animation).

## Install

```bash
npm install three-vrm-utils
```

### Peer dependencies

```bash
npm install react react-dom three @react-three/fiber @pixiv/three-vrm @pixiv/three-vrm-animation
```

For webcam look-at features, also install:

```bash
npm install @mediapipe/tasks-vision
```
